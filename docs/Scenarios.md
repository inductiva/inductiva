# Scenarios

**Inductiva API** provides easy parameterizable simulations of physical systems of interest. These simulations are called **scenarios**. These allows users to choose and iterate over a few parameters and configure the system according to their needs.

Currently, we provide some pre-built scenarios that define physical systems of interest ready to simulate.
