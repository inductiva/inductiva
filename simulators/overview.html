Overview

Overview#

Inductiva API has available several open-source simulators ready to use. Users who are familiar with the simulators can easily start running simulations with their previously prepared simulation configuration files.

As describe in the Simulators overview section typical usage patterns involve:

  • Single executable simulator;

  • Pre-defined executables running in sequence;

  • Choose the executables as wished to run in sequence.

The simulators available in the current version of the API (0.4) are:

  • DualSPHysics

  • SPlisHSPlasH

  • OpenFOAM

  • GROMACS

  • SWASH

  • XBeach

  • Reef3D

  • SWAN

  • FDS

Check the documentation of each simulator to learn about the specifics of how to configure them and launch your simulations via the API.

If you have any questions or suggestions about other open-source simulation packages that you would like to see available via Inductiva API, please open an issue on our GitHub repository, or contact us via support@inductiva.ai.
